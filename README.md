# 🍕 PizzaRecipes Android App

##  Description
PizzaRecipes est une application Android simple permettant d’afficher une liste de pizzas et leurs détails (image, prix, durée, ingrédients, description et étapes de préparation).

L’application est structurée selon une architecture propre en couches :
**DAO / Service / Adapter / UI (Activities)**

## Architecture du projet

1. DAO (Data Access Object)
2. Service (ProduitService)
3. Adapter (PizzaAdapter)
4. Activities (UI)
